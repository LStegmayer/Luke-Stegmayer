Femto-4 is a concept for a tiny fantasy console (hence the name playing on Pico-8) based on a 64x64 output with 8 lovingly-chosen colors, 4x4 1-bit tiles, no scrolling or arbitrary drawing support, and a simple programming language intended to suit the tight constraints. It didn't get very far, as you can see -- just a tile and map editor, which you can use to create mockups. I'd really like to return to this project someday, but time will tell.
